# Netflix Login – Test Cases Portfolio

## Overview
This repository contains **manual test cases** created for the **Netflix login and related public-facing pages**.

The purpose of this project is to demonstrate:
- Manual testing skills
- Test case design and documentation
- Understanding of validation, edge cases, and user flows
- Experience using tools such as Jira & Zephyr

All test cases are written based on **observable behaviors of the public UI only**.

---

## Scope
The test cases cover scenarios such as:
- Email validation (valid, invalid, empty, whitespace, character limits)
- Navigation and redirection behaviors
- Input field behavior and focus handling
- Language selection behavior
- Basic accessibility and usability checks

---

## Tools & Techniques
- Manual Testing
- Test Case Design
- Boundary Value Analysis
- Negative & Positive Scenarios
- Jira + Zephyr (test management)
- CSV-based documentation for GitHub
