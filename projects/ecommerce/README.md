## About This Project

This project contains manual test cases and bug reports created
by executing test scenarios on a sample e-commerce application
(Swag Labs).

Test cases were designed, executed, and documented using
Jira & Zephyr Scale, then exported as CSV files for portfolio purposes.

## Test Scope
- Login
- Inventory
- Add to Cart
- Checkout
- Logout

## Test Artifacts
- Manual test cases written and executed in Zephyr
- Test cases exported as CSV
- Bug reports documented separately

## Test Approach
- Positive & negative scenarios
- UI and functional validation
- Test execution with pass/fail results
